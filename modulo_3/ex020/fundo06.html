ShortHand do background

background: color > url('') > position > repeat > attachment;

Como colocar cor de fundo

background-color: black, blue, rgb(00982f) etc;

Como colocar uma imagem de plano de fundo

background-image: url('');

Como colocar a posicao de referencia da imagem

background-position: x y;

  • Valores para o X mudam o sentido horizontal : left, center, right
  • Valores para o Y mudam o sentido vertical : top, center, bottom

Como escolher com vai se repetir

background-repeat: ;

  • "repeat" vem por padrao, repete durante a pagina toda tanto no eixo X como no Y
  • "no-repeat" faz a imagem aparecer apenas uma vez
  • "repeat-x" faz a imagem a imagem se repetir apenas no eixo X
  • "repeat-y" faz a imagem a imagem se repetir apenas no eixo Y

Valores para bg-size

background-size: ;

  • Por padrao fica 'auto'.
  • Pode ser unidade de medida(ex: 100px, 10vh, 100%).
  • 'Contain' exibe a imagem completa, mas a imagem nao pega toda a pagina.
  • 'Cover' pega a pagina toda, mas nao mostra toda a imagem(estou usando nessa pagina).

Valores para bg-attachment

background-attachment: ;

  • 'fixed' torna a imagem fixa
